# MVP Roadmap

## Phase 1 — Project Setup

Goal:
Create clean folders, documentation, and development base.

Status:
Not started.

## Phase 2 — Open-Source Base

Goal:
Run City Intelligence Cockpit locally.

Tasks:
- clone the open-source repo
- install dependencies
- run locally
- understand folder structure
- remove unnecessary demo layers

## Phase 3 — Munich Data

Goal:
Add first useful Munich data layers.

First categories:
- pharmacies
- offices
- restaurants
- coworking spaces
- universities
- clinics
- shops
- industrial areas

Sources:
- OpenStreetMap
- Overpass API
- GeoJSON exports

## Phase 4 — Filters

Goal:
Allow filtering the city by category.

Filters:
- category
- distance
- saved/not saved
- opportunity score later

## Phase 5 — Lead Saving

Goal:
Click a place and save it as a lead.

Lead fields:
- name
- address
- category
- notes
- score
- status

## Phase 6 — AI Opportunity Scoring

Goal:
AI explains why a place might be interesting.

Prompt:
Analyze this business/location and score its potential for AI automation, procurement tools, local marketing, or business development.

## Phase 7 — Better UI

Goal:
Make the cockpit clean and useful.

UI sections:
- map
- filters
- selected location panel
- saved leads
- AI analysis
