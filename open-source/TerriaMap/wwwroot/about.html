City Intelligence Cockpit

City Intelligence Cockpit is a local-first Munich map workspace for exploring business locations, official public datasets, visual context layers, and Munich 3D dataset footprints. It helps turn map features into saved leads, local opportunity scores, outreach drafts, and JSON or CSV exports.

Local-first by design

This prototype runs locally in the browser. Saved leads use browser localStorage. There is no backend, database, authentication, OpenAI API key, paid API, or deployment requirement.

Data sources

Business discovery layers are generated from OpenStreetMap through Overpass and stored as local GeoJSON. Public dataset layers come from official Munich, Germany, Bavaria, and Europe public-data sources where stable no-key endpoints or metadata are available.

OpenStreetMap and public datasets are useful for discovery but are not guaranteed to be complete, current, or commercially verified. Check important leads manually before outreach.

Open geospatial foundation

The app is built on an open-source geospatial foundation and keeps necessary source, license, and attribution metadata with the catalogue layers.
